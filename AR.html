Aplikasi AR Informasi Gedung

Proyek Augmented Reality untuk menampilkan informasi gedung menggunakan Unity dan Vuforia.

Tentang Proyek

Aplikasi ini menggunakan teknologi Augmented Reality untuk memberikan informasi detail tentang bangunan dan ruangan di FILKOM UB. Dengan memindai marker, pengguna dapat melihat informasi ruangan, gedung, dan fasilitas secara lengkap dan interaktif.

  • ✅ Menggunakan Unity 3D dan Vuforia
  • ✅ Dapat menampilkan informasi dalam bentuk teks dan suara
  • ✅ Dapat memperbesar dan memperkecil panel informasi
  • ✅ Data gedung disimpan dalam file JSON
  • ✅ Informasi dapat diperbarui secara online
